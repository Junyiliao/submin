OpenSSH format

Only OpenSSH format public keys are supported. OpenSSH saves its public keys in ~/.ssh by default, with the extension .pub. Typical names are id_dsa.pub and id_rsa.pub

The contents of this file should look like the following:

  • it starts with 'ssh-rsa', 'ssh-dss', or similar
  • it is then followed by a long string of seemingly random characters
  • finally there is a comment field, usually in the format user@hostname.

For example:

ssh-rsa AAAAB3...(characters omitted for brievity)...d4QSo0qL user@hostname

PEM style ssh key

Sorry, this format is not supported. You need to convert your key to an OpenSSH style key, or export it as such.

An example of this key is:
